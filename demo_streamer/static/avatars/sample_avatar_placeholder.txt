# Sample Avatar Placeholder

Đây là file placeholder cho avatar mẫu.

Để test hệ thống, bạn có thể:

1. Tải một video avatar mẫu từ internet
2. Đặt tên file là: sample_avatar.mp4  
3. Copy vào thư mục này
4. Hoặc dùng chức năng upload trong giao diện admin

Video nên có:
- Một người nói
- Thời lượng 10-30 giây
- Chất lượng HD
- Định dạng MP4
